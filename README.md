# Bike-Buyers-Analysis

## Project Overview
This project analyzes a customer dataset to identify the factors that influence bike purchasing behavior. Using Excel, I performed data exploration, segmentation analysis, and dashboard creation to uncover patterns among customers who purchased bikes versus those who did not.

## Objectives
- Analyze customer demographics and purchasing behavior.
- Identify characteristics of customers most likely to purchase a bike.
- Explore the relationship between income, age, occupation, commute distance, and bike purchases.
